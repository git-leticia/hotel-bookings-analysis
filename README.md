# Dashboard de Business Intelligence para Análise do Comportamento de Cancelamento de Reservas Hoteleiras

## Sobre o Projeto

Este projeto foi desenvolvido como Trabalho de Conclusão do MBA em Business Intelligence.

O objetivo foi desenvolver um dashboard interativo em Power BI para transformar dados de reservas hoteleiras em informações estratégicas, permitindo a análise de  padrões de cancelamento, sazonalidade e comportamento dos clientes.

Você também pode assistir o vídeo da minha apresentação através do link do Youtube: https://youtu.be/y6ATs9q3oLI

## Objetivos

- Analisar o comportamento das reservas.
- Identificar padrões de cancelamento.
- Construir indicadores estratégicos.
- Apoiar a tomada de decisão.

## Ferramentas

- Power BI
- Power Query
- DAX
- Excel

## Indicadores

- Total de Reservas
- Taxa de Cancelamento
- Lead Time
- Tipo de Pagamento

## Principais Insights

- Análise dos fatores relacionados ao cancelamento.
- Descoberta de um comportamento atípico: reservas não reembolsáveis apresentaram uma taxa de cancelamento superior às reservas reembolsáveis.
- Identificação da ausência de informações sobre o motivo do cancelamento, evidenciando uma limitação para análises mais aprofundadas e para a tomada de decisão.
- Dashboard para suporte à tomada de decisão.

## Base de Dados

Hotel Bookings Dataset (Kaggle)
